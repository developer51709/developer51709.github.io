Privacy Policy

last updated: February 24, 2026

1. Data we store

Niko stores only the data required for its features, such as:

  • Leveling stats (XP, level, user ID, guild ID)
  • Configuration data (e.g., uwu‑lock settings)

2. Data we do not store

Niko does not store message content long‑term, except where technically required for a feature (e.g., temporary processing for uwu‑lock).

3. Third‑party services

Niko may call external APIs (e.g., for images or memes). Those services have their own privacy policies.

4. Removal of data

If you want data related to your server removed, you can remove Niko from the server or contact the developer.
