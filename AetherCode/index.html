Everything You Need

A complete development toolkit built for Android devices.

⌨️

Built-In Terminal

A Termux-style terminal embedded directly into the app. Run scripts, package managers, local servers, and common developer workflows without leaving.

Core
📁

Visual File Manager & Editor

Browse, manage, and edit files with ease. Supports a wide range of file types including SQLite databases, JSON, Markdown, and source code.

Core
🤖

AI Tools

Fully optional AI assistant. Supports local models, exposes a local API for other apps, and provides AI-powered crash report summaries.

Optional
📖

Documentation Viewer

In-app viewer for Python and Node.js package documentation. Quick reference without switching apps — works fully offline.

Core
🐙

GitHub Integration

Visual GitHub interface for committing, pushing, and managing repositories. OAuth authentication and full API integration.

Core
👁️

Project Previews

Preview web or app projects directly inside the environment. Useful for rapid prototyping and testing without an external browser.

Core
📶

Offline Mode

Work fully offline with cached runtimes, packages, and documentation. No internet required for your day-to-day workflow.

Core
🧪

Experimental Mode

Toggle experimental features in advanced settings. Access early, unstable, or in-development tools without affecting your main workflow.

Experimental
🪲

Crash Reporting

Detailed crash logs with optional AI-generated summaries. Quickly understand what went wrong and how to fix it.
