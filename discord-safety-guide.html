Threat model in one minute
Start here
Think in three layers:

  • Automation: bots, webhooks, external apps.
  • Surfaces: channels, DMs, voice, threads, reactions.

Quick wins for any server size

  • Turn on 2FA for moderators and owners.
  • Lock powerful permissions to very few roles.
  • Use a logging channel for joins, bans, and role changes.

Server safety checklist
Use this page as a quick audit. If you can honestly check most of these boxes, your server is already ahead of the majority.
Configuration

  • Verification level is set above ‚ÄúNone‚Äù.
